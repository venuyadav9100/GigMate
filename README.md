
# GigMate - The Smart Gig Companion

## Features
- Multi-platform earnings tracking
- AI-powered demand prediction
- Financial management (expense tracking, tax, savings, loans)
- Career optimization and upskilling

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
